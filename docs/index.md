## Общие вопросы

??? question "Что входит в стоимость подписки?"
    За **70 рублей в месяц** вы получаете безлимитный доступ к VPN-серверам и специализированным Telegram-прокси. Дополнительно (по желанию) оплачивается только трафик для обхода региональных «белых списков».

??? question "На скольких устройствах можно использовать VPN?"
    По умолчанию подписка работает на **3 устройствах**. Лимит можно увеличить до 6, 9 или 12 устройств за дополнительную плату (это разовая покупка, лимит увеличивается навсегда). 

??? question "Есть ли бесплатный пробный период?"
    Да, вы можете получить **3 дня бесплатного доступа**. Для этого в боте нажмите `Получить доступ` ➡️ `Пробный доступ`.

## Решение проблем (Troubleshooting)

??? question "Ошибка: «Тайм-аут» / «Host not specified»"
    Это значит, что телефон не может скачать конфигурации.
    
    1. Отключите другие VPN-приложения.
    2. Переключитесь с Wi-Fi на мобильный интернет (или наоборот).
    3. Установите альтернативное приложение (v2RayTun вместо Happ).

??? question "Высокий пинг в приложении. Это нормально?"
    **Да, это нормально.** Приложение использует проверку "via proxy", замеряя полное время ответа. На реальную скорость загрузки файлов это не влияет.
