# Решение проблем

### Ошибка «Тайм-аут» / «Host not specified»
Устройство не может связаться с сервером для загрузки конфигураций.
1. Отключите другие VPN-приложения.
2. Смените сеть (с мобильного интернета на Wi-Fi или наоборот).
3. Используйте альтернативное приложение (например, v2RayTun вместо Happ).

### VPN подключен, но сайты не открываются
1. Проверьте пинг в приложении. Наличие цифр означает, что сервер работает.
2. Проверьте настройки приложения и отключите лишние ползунки.
3. Смените локацию сервера.
4. Отключите программы обхода блокировок на ПК (GoodbyeDPI / Zapret), так как они могут конфликтовать с клиентом.
5. Проверьте доступность интернета без VPN. Если обычные сайты не грузятся, ваш оператор включил "белые списки" — вам понадобится соответствующий сервер.

### Высокий пинг в приложении
Это нормальное поведение. Приложение Happ использует тип проверки "via proxy", который замеряет полное время ответа. Это не влияет на реальную скорость соединения. В играх пинг будет стандартным (20-90 мс).

### Блокировка в ИИ-сервисах (Google, Gemini)
Google периодически блокирует IP-адреса дата-центров. Мы маршрутизируем трафик через Cloudflare для решения проблемы. Если доступ закрыт на одной локации, просто выберите сервер другой страны в приложении.

### Telegram-прокси не работает / висит «Подключение»
Из-за блокировок работа прокси может быть нестабильной.
* Покликайте по прокси в списке вручную, чтобы обновить статус.
* Включите настройку «Автопереключение прокси».
* Используйте прокси со словом **relay** в названии.
* В качестве альтернативы включите основной VPN — Telegram будет работать через него.

### Задержка после оплаты
Платежные шлюзы могут обрабатывать транзакции с небольшой задержкой. Если после оплаты подписка не обновилась в течение часа, обратитесь в поддержку.
